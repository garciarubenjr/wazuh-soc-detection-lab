# wazuh-soc-detection-lab
## Wazuh SOC Detection Lab

This project demonstrates a real-world SOC detection environment using Wazuh as a SIEM/HIDS platform.

### Features
- Endpoint agent enrollment
- File integrity monitoring
- Privileged access auditing
- Network port change detection
- CIS compliance auditing

### Tools
- Wazuh 4.14
- Ubuntu 24.04
- VMware

### Skills Demonstrated
- SIEM deployment and tuning
- SOC alert analysis
- MITRE ATT&CK mapping
- Compliance assessment
- Incident reporting
